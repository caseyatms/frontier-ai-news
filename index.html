ℹ️ How refresh works. This is a static dashboard whose data lives in data.js. Two ways to update it: (1) In-browser — click ⚙ Settings, paste your own AI API key (Anthropic, OpenAI, or xAI), then ⚡ Refresh now; the page asks that model for the latest news. Default is use-once (key held in memory only, never saved); you can opt to save it with a daily cap. Your key is sent only to the provider you chose — never committed or shared. (2) Claude Code — say "refresh the AI dashboard". Always confirm fast-moving details against the linked primary sources.
